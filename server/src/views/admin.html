📤 Import XML

Charger un fichier XML dans la base de connaissance

Nouveau fichier XML


      


      

        

          
💡 Aide rapide

        

        

          1. Entrez le tenant_id = le groupe de ce fichier (ex: garage_dupont, immo_paris)

          2. Ce tenant doit correspondre a une entree dans WIDGET_TENANT_MAP de votre .env

          3. Dry-run = preview sans ecrire en base / Commit = ecriture reelle

          4. Formats XML supportes : <catalog><listing>, <annonces_immobilieres><annonce>, <properties><property>, format <bien>

          5. Apres commit, les donnees apparaissent immediatement dans l'onglet Proprietes
