# Documentation OracleSentinel

OracleSentinel déploie, pour chaque agence (immobilière ou autre), un **chatbot IA** qui qualifie les leads et les pousse automatiquement dans le **CRM propre à l'agence** — le tout piloté depuis un **QG** (Command Center) et facturé à l'abonnement.

## Sommaire

- [Onboarding d'une nouvelle agence](./onboarding-agence.md) — informations à collecter, mise en service en 1 clic (widget_id + snippet d'intégration), configuration du CRM par agence, cycle de vie (active / suspended / archived).
- [Tarification](./pricing.md) — plans Starter / Pro / Scale, quotas mensuels et facturation Stripe (optionnelle, désactivable).
